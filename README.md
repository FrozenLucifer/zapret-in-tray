# ZapretTray

Простой клиент и трей-меню для  
**zapret-discord-youtube** от Flowseal  
(https://github.com/Flowseal/zapret-discord-youtube)

## Описание
Совместим с актуальной версией **zapret**, так как использует оригинальный репозиторий внутри себя  
(репозиторий скачивается при первом запуске, поддерживается автообновление).

На данный момент реализованы не все функции.  
Дополнительную настройку можно выполнить через `service.bat`, открыв его из трея.
